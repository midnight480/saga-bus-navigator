## ✅ 基本動作確認

### 1. アクセス確認

- [ ] HTTPSでアクセスできる
- [ ] ページタイトルが「佐賀バスナビゲーター」と表示される
- [ ] ファビコンが表示される
- [ ] ローディング表示が出る
- [ ] データ読み込み完了後、UIが有効になる

**確認方法**:
```
1. ブラウザで https://saga-bus.midnight480.com にアクセス
2. ページが正しく表示されることを確認
3. 開発者ツールのコンソールでエラーがないことを確認
```

### 2. データ読み込み確認

- [ ] データ読み込みが3秒以内に完了する
- [ ] エラーメッセージが表示されない
- [ ] 検索フォームが有効になる

**確認方法**:
```
1. ページを開く
2. ストップウォッチで読み込み時間を計測
3. 開発者ツールのNetworkタブでCSVファイルの読み込みを確認
   - bus_stop.csv
   - timetable_all_complete.csv
   - fare_major_routes.csv
```

## 🔍 検索機能確認

### 3. バス停検索（インクリメンタルサーチ）

#### 乗車バス停

- [ ] 「佐賀」と入力すると候補が表示される
- [ ] 「佐賀駅バスセンター」を選択できる
- [ ] 選択後、入力フィールドに反映される

#### 降車バス停

- [ ] 「大学」と入力すると候補が表示される
- [ ] 「佐賀大学」を選択できる
- [ ] 選択後、入力フィールドに反映される

**確認方法**:
```
1. 乗車バス停フィールドに「佐賀」と入力
2. 候補リストが表示されることを確認
3. 「佐賀駅バスセンター」をクリック
4. 降車バス停フィールドに「大学」と入力
5. 「佐賀大学」を選択
```

### 4. 時刻選択

- [ ] 「今すぐ」ラジオボタンが選択できる
- [ ] 「出発時刻指定」を選択するとタイムピッカーが表示される
- [ ] 「到着時刻指定」を選択するとタイムピッカーが表示される
- [ ] 「始発」ラジオボタンが選択できる
- [ ] 「終電」ラジオボタンが選択できる

**確認方法**:
```
1. 各ラジオボタンをクリック
2. タイムピッカーが適切に表示されることを確認
3. 時刻を選択できることを確認
```

### 5. 検索実行

#### 正常系テスト

**テストケース1: 佐賀駅バスセンター → 佐賀大学**

- [ ] 検索ボタンをクリックできる
- [ ] 検索結果が2秒以内に表示される
- [ ] 検索結果に以下の情報が含まれる：
  - [ ] 出発時刻
  - [ ] 到着時刻
  - [ ] 所要時間
  - [ ] 運賃（大人・子供）
  - [ ] 事業者名
  - [ ] 路線名

**テストケース2: 佐賀駅バスセンター → 県庁前**

- [ ] 検索結果が表示される
- [ ] 複数の便が表示される
- [ ] 時刻順にソートされている

**テストケース3: 佐賀駅バスセンター → 西与賀**

- [ ] 検索結果が表示される
- [ ] 運賃情報が正しく表示される

#### 異常系テスト

**テストケース4: 乗車バス停未選択**

- [ ] エラーメッセージ「乗車バス停を選択してください」が表示される
- [ ] 検索が実行されない

**テストケース5: 降車バス停未選択**

- [ ] エラーメッセージ「降車バス停を選択してください」が表示される
- [ ] 検索が実行されない

**テストケース6: 同一バス停選択**

- [ ] エラーメッセージが表示される
- [ ] 検索が実行されない

**テストケース7: 該当便なし（深夜時刻指定）**

- [ ] 「該当する便が見つかりませんでした」メッセージが表示される
- [ ] エラーではなく、正常な結果として表示される
