# 佐賀市内のバスナビゲーションアプリ

## 概要

佐賀市内を走る複数事業者の路線バスについて、
- バス停位置
- 路線情報
- 時刻表
- 運賃
- （可能であれば）現在位置・遅延情報

を横断的に検索・表示できるナビゲーションアプリを提供する。  
地図表示には Google Maps を使わず、OpenStreetMap を利用する。

初期対応は、佐賀市内を走る以下の事業者とする。

- 佐賀市営バス ✅ 完了（9路線、714件の時刻データ）

- 西鉄バス（佐賀エリア） ✅ 完了（3路線、104件の時刻データ）

将来的な拡張候補：
- 昭和バス
- JR九州バス

## データ要件

### 1. バス停位置情報 ✅ 完了

各事業者の全バス停について、以下を保持する。

- バス停名（正式名称）
- 緯度・経度（OpenStreetMap へのマッピングを前定）
- 事業者区分（佐賀市営 / 祐徳 / 西鉄 など）
- 所属路線一覧（そのバス停を通過・発着する路線 ID）
- 必要に応じて、バス停コード（事業者ごとの内部コードがあれば保持）

**実装状況**: `bus_stop.csv` に110件のバス停データを格納（緯度・経度付き）

### 2. 路線情報 ✅ 完了

路線ごとに以下を保持する。

- 路線 ID
- 路線名（例：佐賀駅〜川副線）
- 運行事業者
- 経由バス停の順序付き一覧
- 運行曜日区分（平日 / 土日祝 / 特殊ダイヤ 等）

**実装状況**: 
- `bus_route_saga-city.csv` - 佐賀市営バス28路線
- `bus_route_yutoku.csv` - 祐徳バス15路線
- `bus_route_nishitetsu.csv` - 西鉄バス3路線

### 3. 時刻表情報 ✅ 完了

路線 × バス停 × 曜日区分ごとの時刻表を保持する。

- route_id（路線 ID）
- stop_id（バス停 ID）
- departure_time（そのバス停を発車する時刻）
- direction（上り / 下り 等）
- weekday_type（平日 / 土日祝 / 特殊日 等）

**実装状況**: 
- `timetable_all_complete.csv` - 統合時刻表（1,064件）
  - 佐賀市営バス: 714件（9路線、途中バス停含む）
  - 祐徳バス: 246件（4路線）
  - 西鉄バス: 104件（3路線）
- 便ID、バス停順序により途中バス停間の検索が可能

### 4. 運賃情報 ✅ 完了

- 乗車バス停・降車バス停・事業者・経路に応じた運賃情報を保持／算出する。
- 運賃は **検索実行時点** のものを表示する。
- 検索実行時点は、`ntp.nict.jp` に問い合わせた正確な時刻を基準とする。
  - 端末のローカル時刻に依存しない。
  - 将来的な運賃改定が発生しても、**検索時点** の正しい結果を返すことを意図する。

**実装状況**:
- `fare_info.csv` - 距離帯別運賃表（14件）
- `fare_major_routes.csv` - 主要区間運賃（10件）

### 5. 乗り換え情報 ✅ 完了

**実装状況**:
- `transfer_info.csv` - 乗り換え可能バス停（4箇所）
- `walking_transfer.csv` - 徒歩乗り換え（2箇所）

## 機能要件

### 共通要件

- 地図表示には OpenStreetMap を利用する（Google Maps 無料枠には依存しない）。
- 地図上にバス停をピン表示し、タップ／クリックで選択できる。
- 検索条件として「乗車バス停」「降車バス停」「乗車時刻」を必ず指定する。

### 1. バス停名からの検索

ユーザーは以下を入力して検索できる。

- 乗車バス停名
- 降車バス停名
- 乗車時刻
  - 今すぐ
  - 時刻指定
  - 終電（当日の最終便）

#### 挙動イメージ

- バス停名はインクリメンタルサーチまたは候補一覧から選択可能。
- 「今すぐ」を指定した場合は、`ntp.nict.jp` から取得した現在時刻以降の便を検索する。
- 「終電」を指定した場合は、当日中の最終便を検索する。

### 2. 地図からの検索

ユーザーは地図上で以下を操作して検索できる。

1. OpenStreetMap 上に表示されたバス停ピンから、乗車バス停を選択
2. 同様に、降車バス停を選択
3. 乗車時刻（今すぐ / 時刻指定 / 終電）を指定して検索

### 3. 住所からの検索

ユーザーはテキスト入力で住所を指定し、最寄りバス停を検索できる。

1. 住所を入力
2. 住所から緯度・経度を取得し、一定距離内のバス停を検索
3. 最寄りバス停を候補として提示し、乗車バス停として選択
4. 降車バス停をバス停名検索または地図から選択
5. 乗車時刻（今すぐ / 時刻指定 / 終電）を指定して検索

## 検索結果表示要件

検索結果の一覧では、1便ごとに以下を表示する。

- 乗車時刻（○:○○ 発）
- 到着予定時刻（○:○○ 着）
- 所要時間
- 運賃（検索時点の料金）
- 運行事業者（佐賀市営 / 祐徳 / 西鉄 等）
- 路線名・系統番号
- （取得できる場合）現在のバス位置と遅延情報
  - 何停前を通過済みか
  - 遅延○分 など

## リアルタイム位置情報・遅延情報（将来的要件）

- 佐賀市営バスは、バス車両に搭載された GPS 情報を利用したバスロケーションシステムを導入済み。
  - スマートフォンアプリ「PINA」「楽PINA」、時刻・経路検索アプリ「MOKUIK（もくいく）」から利用可能。
- Google マップでは、佐賀県内を走るバス（佐賀市営バス、祐徳バス、昭和バス、西鉄バス、JR九州バス）の時刻や経路検索、および一部の遅延情報に対応済み。

本アプリでは将来的に、以下を目指す：

- 対応事業者のバス現在位置・遅延情報を取得し、
  - 検索結果一覧に「遅延○分」「何停前を通過」などを表示
  - マップ上でバスアイコンとして現在位置を描画
- 一度検索した便を「ついせき」登録し、終点到着まで以下を追跡表示：
  - 現在位置
  - 遅延状況
  - 予想到着時刻

- **ホーム画面への追加**

- **自動アップデート**：新しいデータが利用可能になると自動更新
